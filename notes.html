Different Html elements:

    Header:
      Uses "h1" to "h6" to create different size headings for websites
    Unorganized list:
      Uses "ul" to create the outline containing different kinds of list elements
    Paragraph:
      Uses "p" as a tag for ceating a block of text
    Article:
      "article" is used for organizing a larger group of text usually more that one paragraph
    href:
      "href" Is used for linking a asset to another html asset, usually known as the "target"
